Matching Game

In this game the user needs to match all the symbols to win. The game randomly shuffles the cards. Users should not be able to open a card that is already open or matched.The game displays a star rating (from 1 to at least 3) that reflects the player's performance. At the beginning of a game, it shows 3 stars. After some number of moves, it changes to a lower star rating. After a few more moves, it changes to a even lower star rating (down to 1). The game was created using JavaScript basics, HTML and CSS

Pig Game

If the player rolls a 1, all his ROUND score gets lost. After that, it's the next player's turn. The player can choose to Hold, which means that his ROUND score gets added to his GLBAL score. After that, it's the next player's turn. The first player to reach 100 points on GLOBAL score wins the game.

Classic Arcade Game

In this game you have a Player and Enemies (Bugs). The goal of the player is to reach the water, without colliding into any one of the enemies. The player can move left, right, up and down. The enemies move in varying speeds on the paved block portion of the scene. Once a the player collides with an enemy, the game is reset and the player moves back to the start square. Once the player reaches the water the game is won.

Budget Calculator

A basic budget calculator webpage built using Object Oriented JavaScript. It allows you to store your income and expenses and also calculates the percentage of both. Easy to use and manage your budget. Just make a note of your income or expenses, enter the value and hit submit. It will add your income or expenses according to the input and store it into the respective field.
